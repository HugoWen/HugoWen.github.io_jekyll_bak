###hugowen.com
Jekyll主题结合了Henter的<a href="http://henter.diandian.com">Henter点点博客</a>和<a href="http://www.pizn.net/14-11-2012/theone-blog-theme/" title="theOne">Jekyll 博客主题 theOne</a>

##Install
```
bundle install
```

###New post
```
./new your-title

或

rake new_post[your-title]

如果你在用zsh，需要转义一下..

rake new_post\[your-title\]

```
